Ensure that when your posts are shared on FaceBook that the proper title, thumbnail, and description is used.

== Description ==

Ensure that when your posts are shared on FaceBook that the proper title, thumbnail, and description is used by setting a couple of extra meta tags on the page.

== Installation ==

1. Download the fbmetatags.zip file, unzip and upload the whole directory to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress.
1. Edit the plugin options to set the default thumbnail image. Everything else is done automatically.
